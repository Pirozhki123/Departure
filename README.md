# <Departure>

## サイト概要
### サイトテーマ
あなたの旅立ちを手助けする写真SNS

### テーマを選んだ理由
旅行に行きたいけど、どんな場所が良いか思いつかない。
過去訪れた場所と似たような場所に行ってみたいけど、なかなか見つからない。
自分が感じた上記のような悩みを解決出来るといいなと思い、このテーマにした。

### ターゲットユーザ
20~50代
ネットが扱えてかつ、ある程度気軽に旅行に行ける層

### 主な利用シーン
旅行の計画時
旅行中
